# 🎨 XEngineer - AI 语音绘图工具

一款纯语音控制的智能绘图工具，用户无需使用鼠标或键盘，仅通过语音指令即可完成绘图创作。

## ✨ 功能特性

### 🎤 语音控制
- **实时语音识别**：基于 Web Speech API，支持中文语音输入
- **语音反馈**：操作完成后自动语音播报结果
- **连续识别**：支持连续语音输入，无需重复点击

### 🤖 AI 智能解析
- **接入阿里云通义千问 API**：智能理解用户意图
- **复杂指令拆解**：支持多步骤绘图指令
- **智能绘图**：可绘制任意物体（如芒果、太阳、房子等）

### 🎯 精准定位绘图
- **坐标定位**：支持指定坐标绘制形状
- **边框坐标标签**：方便用户定位画布位置
- **多种形状**：圆形、矩形、三角形、椭圆、直线、星星、心形

### 🎨 丰富的绘图功能
- **自定义调色盘**：支持任意颜色选择
- **预设颜色**：12 种常用颜色快速切换
- **画笔大小调节**：支持 1-50 像素
- **填充/描边切换**：支持形状填充和描边模式
- **撤销/重做**：支持历史记录管理

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Edge、Safari 等）
- 麦克风权限
- 网络连接（用于 AI 解析）

### 安装与运行

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/XEngineer-AI-voice-drawing-tool.git
   ```

2. **配置 API Key**
   
   打开 `src/config/ai.js`，填入你的阿里云 API Key：
   ```javascript
   export const AI_CONFIG = {
     apiKey: 'your-api-key-here',
     // ...
   };
   ```

3. **运行项目**
   
   直接用浏览器打开 `index.html` 即可运行。

### 获取 API Key

1. 访问 [阿里云百炼平台](https://dashscope.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 开通百炼服务
4. 在 API Key 管理页面创建 API Key

## 🔧 技术栈

- **前端框架**：原生 JavaScript (ES6+)

- **绘图引擎**：Canvas API
- **AI 服务**：阿里云通义千问 (qwen-turbo)
- **样式**：CSS3 响应式设计
