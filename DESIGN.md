# 语音控制绘图工具 - 设计文档

## 1. 项目概述

### 1.1 项目目标
开发一款纯语音控制的智能绘图工具，用户无需使用鼠标或键盘，仅通过语音指令即可完成绘图创作。集成 AI 模型实现智能指令理解和任意物体绘制。

### 1.2 核心要求
| 指标 | 要求 | 实现状态 |
|------|------|----------|
| **指令准确性** | 准确识别和理解用户的语音指令 | ✅ 已实现（本地解析 + AI 解析） |
| **容错性** | 对不同表达方式和口音具有一定容忍度 | ✅ 已实现（AI 模型支持） |
| **响应延迟** | 语音到绘图操作的响应时间 < 500ms | ⚠️ 部分实现（本地指令快，AI 指令约 2-3s） |
| **复杂指令处理** | 支持复杂指令的拆解与执行 | ✅ 已实现（多步指令支持） |
| **智能绘图** | 绘制任意物体（如芒果、太阳等） | ✅ 已实现（AI 智能绘图） |
| **图形修改** | 支持对已绘制图形的选择、修改和删除 | ✅ 已实现 |

### 1.3 应用场景
- 无障碍人士绘图创作
- 智能设备端的免触控操作
- 创意设计的语音交互体验
- 教育场景中的语音教学
- AI 辅助的智能创作

## 2. 技术架构

### 2.1 技术选型

| 分类 | 技术 | 版本 | 选型理由 |
|------|------|------|----------|
| 绘图引擎 | HTML5 Canvas API | - | 浏览器原生支持，性能优异 |
| 语音识别 | Web Speech API | - | 浏览器原生支持，无需额外依赖 |
| 语音合成 | Web Speech API | - | 浏览器原生支持，轻量高效 |
| AI 服务 | 阿里云通义千问 | qwen-turbo | 免费试用，中文理解能力强 |
| 语言 | Vanilla JavaScript | ES6+ | 无框架依赖，加载速度快 |
| 样式 | CSS3 | - | 现代CSS特性，响应式设计 |

### 3.3 指令解析流程

```
语音输入 → 语音识别 → 文本预处理 → 本地解析 → AI 解析 → 指令执行 → 结果反馈
    ↓              ↓              ↓           ↓          ↓          ↓          ↓
  麦克风       Web Speech     去除噪声     关键词匹配  通义千问API  绘图操作   语音播报
                              ↓           ↓          ↓
                           格式化文本   简单指令    复杂指令
                              ↓           ↓          ↓
                           标准化处理   快速响应    智能理解
```

### 3.4 AI 指令解析机制

#### 3.4.1 解析模式

| 模式 | 适用场景 | 响应时间 | 示例 |
|------|----------|----------|------|
| 本地解析 | 简单预设指令 | < 100ms | "撤销"、"红色"、"画笔" |
| AI 解析 | 复杂组合指令 | 2-3s | "画一个黄色的五角星"、"画一个芒果" |

#### 3.4.2 AI 解析流程

```
用户指令 → 通义千问 API → JSON 指令 → 执行引擎 → 绘图结果
    ↓              ↓              ↓           ↓          ↓
"画一个芒果"   智能理解       smartDraw    多步执行    画布显示
                ↓              ↓           ↓
              生成步骤       steps数组    椭圆+弧线
```

#### 3.4.3 AI 输出格式

```json
// 简单指令
{"action": "setColor", "params": {"color": "#ff0000"}}

// 绘制形状
{"action": "draw", "params": {"shape": "circle", "x": 200, "y": 500, "radius": 50}}

// 多步指令
{"action": "multiStep", "params": {"steps": [
  {"action": "setColor", "params": {"color": "#ffff00"}},
  {"action": "draw", "params": {"shape": "star", "x": 400, "y": 300, "size": 50}}
]}}

// 智能绘图
{"action": "smartDraw", "params": {"object": "芒果", "x": 400, "y": 300, "steps": [
  {"type": "ellipse", "x": 0, "y": 0, "width": 60, "height": 100, "color": "#ffa500", "filled": true},
  {"type": "arc", "x": 0, "y": -20, "radius": 30, "startAngle": 200, "endAngle": 340, "color": "#ff8c00"}
]}}

// 图形修改
{"action": "selectFirstObject", "params": {}}
{"action": "modifyColor", "params": {"color": "#ff0000"}}
{"action": "moveObject", "params": {"direction": "right", "distance": 50}}
{"action": "deleteSelectedObject", "params": {}}
```
